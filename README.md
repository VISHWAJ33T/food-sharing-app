This is the boilerplate/starter kit for next js web apps.

## How to use this template?

[Refer Image] Click on `Use this template` > `Create a new Repository`

![alt text](image.png)

## Integrations

- Auth setup `next-auth`
- Tanstack Query
- SWR Enabled
- Redux
- MUI & Emotion
- Tailwind CSS
- MDX Editor
- File Upload SDK
- TypeScript
- Eslint + Prettier
- Firebase
- Prisma + MongoDB
